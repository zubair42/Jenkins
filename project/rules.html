RULES

Rule 1RULES. Only registered users can play the game.
Rule 2.  Registration is free.
Rule 3. The game will be played between two players
Rule 4.  Before starting the game both the users must login
Rule 5.  If the user has earlier played the game then his profile will show his highest score and his current level so far.
Rule 6 . A player can only advance to higher level only if he has completed all previous stages successfully.
Rule 7.  A player must start firing within the specified time limit,if due to any reason he fails then his opponent will be declared as a winner
